## 列表项属性命名规范
| 为实现列表的动态化，需要对列表项属性命令进行标准化定义

| 属性名称     | 类型     | 说明     |
| ----------- | -------- | -------- |
| avator      | Url      | 头像     |
| image       | Url      | 尺寸较大图片 |
| logo        | Url      | 尺寸较小大图标 |
| title       | String   | 标题     |
| subtitle    | String   | 副标题   |
| body        | String   | 详情内容 |
| tags        | Array    | 标题(title)右侧标签  |
| timestamp   | Datetime | 标题靠右时间戳  |
