## 列表项属性命名规范
| 为实现列表的动态化，需要对列表项属性命令进行标准化定义

| 属性名称     | 类型     | 说明     |
| ----------- | -------- | -------- |
| type        | String   | 类型     |
| avatar      | Url      | 头像     |
| image       | Url      | 尺寸较大图片 |
| icon        | Url      | 尺寸较小大图标 |
| title       | String   | 标题     |
| subtitle    | String   | 副标题   |
| body        | String   | 详情内容 |
| tags        | Array    | 标题右侧标签(title.tags) |
| timestamp   | Datetime | 标题靠右时间戳  |
| desc        | String   | 描述     |
